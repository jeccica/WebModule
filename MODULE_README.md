# How to use

```js
<script src="lib/Zzz.js">
<script>
// for Browser
console.log( Zzz() );
</script>
```

```js
// for WebWorkers
importScripts("lib/Zzz.js");

console.log( Zzz() );
```

```js
// for Node.js
var Zzz = require("lib/Zzz.js");

console.log( Zzz() );
```
