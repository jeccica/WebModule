# How to use

## Browser

```js
<script src="lib/Zzz.js">
<script>
console.log( Zzz() );
</script>
```

## WebWorkers

```js
importScripts("lib/Zzz.js");

console.log( Zzz() );
```

## Node.js

```js
var Zzz = require("lib/Zzz.js");

console.log( Zzz() );
```
